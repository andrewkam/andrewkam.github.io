MusicBrainz

About

Founded by Robert Kaye, MusicBrainz has since grown rapidly from a one-man operation to an international community of enthusiasts who appreciate both music and music metadata. Along the way, the scope of the project has expanded from its origins as a mere a CDDB replacement to today, where MusicBrainz has become a true encyclopedia of music.

Overview

  • Community-maintained open source encyclopedia of music information
  • Anyone allowed to contribute
  • No discrimination of any type of music over another

History

  • CDDB, database for applications to query CD information, commercialized in 2000
  • Founded by Robert Kaye as a replacement
  • Operated by the MetaBrainz Foundation, a California-based non-profit corporation

Architecture

  • Database
    • PostgreSQL
    • Contains all metadata and history
    • Schema
  • Application Server
    • Provides main access through website
    • Provides web service

Usage

  • Contribute via website following guidelines
  • Commit modifications through software clients
